# 康宝莱中国市场专项研究报告

> 版本日期：2026-05-29  
> 用途：商业判断、合作/代理项目评估、合规风险识别  
> 重要提示：本文不是法律意见、投资建议或医学建议。正式落地项目前，应由律师、财税顾问和当地市场监管/商务主管部门口径进行复核。

## 一、结论先行

康宝莱在中国大陆不是“无牌代理项目”，而是有直销经营许可、服务网点和直销产品公示的直销企业。根据商务部直销行业管理系统公示信息，康宝莱（中国）保健品有限公司目前显示有 34 个分支机构、2504 个服务网点、6 类 170 种直销产品、451 名在册直销培训员。

但中国市场不能简单套用海外的 MLM 或多层级经销逻辑。中国大陆直销监管对经营区域、服务网点、产品范围、直销员身份、计酬制度、培训宣传和退换货都有明确边界。康宝莱中国公开披露的直销员计酬制度也强调：报酬只能按本人直接向消费者推销产品的收入计算，最高不超过 30%，不按介绍顾客人数计酬，不因介绍顾客获得报酬，并禁止团队计酬。

从商业表现看，康宝莱中国区目前不是高增长状态。Herbalife 2025 年 10-K 显示，中国区 2025 年净销售额约 2.791 亿美元，较 2024 年下降 6.2%，约占全球净销售额 6%。公司在中国继续投入数字工具、客户忠诚计划、营养俱乐部和产品扩展，但短期仍面临销量下滑、监管趋严、直销行业声誉和消费者信任等压力。

如果把它作为项目来评估，核心不是“能不能卖康宝莱”，而是必须先回答四个问题：是否在获批区域内做、是否只卖获批产品、参与者身份和计酬是否合法、宣传和私域运营是否避开传销/虚假宣传/医疗功效宣称红线。

## 二、康宝莱中国概况

康宝莱是一家全球营养和健康产品公司，核心产品覆盖体重管理、日常营养、运动营养、能量与健康生活方式等方向。中国大陆业务主体主要涉及康宝莱（中国）保健品有限公司及康宝莱在华相关实体。

康宝莱中国官网披露，公司在中国市场强调“营养 + 运动”的健康生活方式，并把营养俱乐部、数字化平台、官方商城和营销人员赋能工具作为中国业务的重要组成部分。官网资料显示，康宝莱中国于 2019 年启动中国区业务数字化转型，提出“双中台、三中心”生态蓝图，包括业务中台、数据中台，以及面向营销人员、顾客和未来创新的中心能力。

### 中国业务的关键特征

| 维度 | 说明 |
|---|---|
| 法律属性 | 有直销经营许可的直销企业，但经营必须受直销监管约束 |
| 主要场景 | 直销产品销售、营养俱乐部、线上商城、社交电商、营销人员服务 |
| 主要产品方向 | 营养补充、体重管理、运动健康、健康生活方式相关产品 |
| 关键渠道 | 直销员/营销人员、服务提供商、营养俱乐部、官方线上工具 |
| 合规核心 | 区域、产品、身份、计酬、宣传、退换货、数据合规 |

## 三、中国直销许可与经营边界

中国大陆直销不是“全国随便做”的销售模式，而是许可 + 区域 + 服务网点 + 产品清单的组合监管。

商务部直销行业管理系统常见问题页面明确：企业从事直销经营必须经过商务部批准获得直销经营许可；未经批准，任何单位和个人不得从事直销。直销企业也不能自动在全国开展直销业务，必须在拟从事直销活动的省、自治区、直辖市设立分支机构，并在开展直销活动的地区建立服务网点。未经审核公布的地区不得开展直销业务。

对康宝莱中国而言，商务部系统公示信息是判断其合法直销范围的主依据。当前公示口径包括：

| 项目 | 当前公示信息 |
|---|---:|
| 分支机构 | 34 个 |
| 服务网点 | 2504 个 |
| 直销产品 | 6 类 170 种 |
| 直销培训员 | 451 人在册 |

这意味着：即使康宝莱在中国有直销许可，具体项目仍需逐项核查当地是否在获批区域内、是否有服务网点、具体销售产品是否属于商务部公示直销产品、参与人员是否具备合规身份。

## 四、中国产品体系

康宝莱中国的产品不能只按“公司官网有什么”来判断，还必须分成两层：

1. **商务部公示的直销产品**：这些才是可以通过直销方式销售的产品，应以商务部直销行业管理系统和康宝莱直销信息披露网站为准。
2. **官网展示或其他普通销售产品**：可以用于了解品牌产品矩阵，但不能当然等同于“可直销产品”。

从产品方向看，康宝莱在中国的核心仍围绕营养健康和体重管理展开，典型卖点包括代餐、蛋白、营养补充、运动与健康生活方式管理等。项目评估时应额外核查每个 SKU 的保健食品注册/备案、普通食品属性、标签、广告可宣称范围，以及是否被纳入直销产品目录。

### 产品合规判断

| 问题 | 合规判断口径 |
|---|---|
| 能否通过直销卖？ | 是否在商务部/企业披露的直销产品目录内 |
| 能否宣称减肥、治疗、预防疾病？ | 通常不能作医疗化、疾病治疗/预防宣称；保健功能也必须限于批准/备案范围 |
| 能否低价转卖或跨平台倒货？ | 康宝莱官网提示产品按公司市场零售价销售，低价渠道存在质量和权益风险 |
| 能否做“药品替代”？ | 康宝莱顾客保障条款明确产品并非药物，不能替代药物 |

## 五、中国经销/服务商模式详解

### 1. 中国模式与海外模式的关键区别

海外市场中，Herbalife 通常以独立经销商、会员、自用折扣、零售利润、团队销售活动相关佣金等方式组织销售网络。但中国大陆监管环境不同，直销与传销边界更敏感，中国业务不能简单表述为“发展下线拿团队佣金”。

Herbalife 2025 年 10-K 也披露，中国业务需要采用与其他国家不同的 modified business model，并称中国业务涉及公司运营的零售平台、社交电商、销售代表和独立服务提供商等安排。公司同时提示，中国市场存在行业特定监管、法律解释和执法不确定性。

### 2. 直销员/业务代表

康宝莱中国直销员推销合同显示，直销员需通过直销业务培训和考试，签订推销合同，并取得直销员证后方可开展直销活动。合同也列明若干身份限制：年满 18 周岁、具有完全民事行为能力的中国公民；不得为教师、医务人员、公务员、现役军人、全日制在校学生、境外人员、其他直销企业正式员工、康宝莱员工等。

直销员的经营边界包括：

| 边界 | 要求 |
|---|---|
| 区域 | 只可在商务主管部门批准的直销活动区域内开展 |
| 产品 | 只可推销获批直销产品，不得推销未列入批准清单的其他产品 |
| 身份展示 | 向消费者推销时应出示直销员证和推销合同 |
| 价格与凭证 | 按产品标明价格推销，成交后提供发票/售货凭证和退换货信息 |
| 行为限制 | 不得强行推销、误导宣传、夸大产品功能、从事传销或变相传销 |

### 3. 计酬制度

康宝莱中国披露的直销员计酬制度是合规判断的核心。其公开页面列明：

| 月销售金额 | 销售报酬比率 |
|---:|---:|
| 1201-6000 元 | 15% |
| 6001-24000 元 | 20% |
| 24001-48000 元 | 25% |
| 48001 元及以上 | 30% |

计酬原则包括：

- 销售报酬按本人直接向消费者推销产品的收入计算。
- 销售报酬总额不超过本人直接向消费者推销产品收入的 30%。
- 不以介绍顾客人数作为计算报酬依据。
- 不因介绍顾客行为获得报酬。
- 禁止团队计酬。

所以，如果某个项目话术强调“拉人头”“层级返点”“团队无限代数收益”“购买礼包成为上级”“发展多少人即可躺赚”，就应视为重大合规风险，不能用“康宝莱有直销牌照”来掩盖。

### 4. 服务提供商与营养俱乐部

康宝莱中国顾客保障条款明确，康宝莱服务提供商是独立业务承包商，不是康宝莱（中国）员工。康宝莱中国官网将营养俱乐部描述为由服务提供商运营的社交空间，提供一对一营养健康知识和运动方式服务。

营养俱乐部的商业价值在于：

- 提供线下触点，降低营养品的体验门槛。
- 通过社群、打卡、活动和体重管理挑战增强复购。
- 为营销人员提供与客户持续互动的场景。
- 结合官方商城、App、小程序等数字工具进行订单、服务和客户管理。

但营养俱乐部也是合规敏感点。需要重点核查是否涉及未获许可经营、食品制售资质、虚假宣传、暗示治疗疾病、诱导囤货、以活动包装招募、低价/跨平台转卖、个人信息过度收集等问题。

### 5. 线上商城、社交电商与私域

康宝莱中国官网披露，其数字化成果包括基于微信端的“康宝莱官方商城”、业务中台和新“莱聚”App。公司隐私条款也显示，康宝莱可能将顾客订单、身份、积分、健康目标、上秤记录、健康报告、活动打卡等信息以适当形式分享给营销人员，以便提供营养咨询、产品咨询、体重管理等服务。

这说明中国业务已经不是纯线下直销，而是“线下服务 + 官方数字工具 + 私域运营”的复合模式。它的机会在于数据化复购和服务留痕，风险在于个人信息保护、健康数据处理、营销授权、客户画像、活动排行榜和社群传播内容的合规。

## 六、市场数据与趋势

Herbalife 2025 年 10-K 披露，2025 年全球净销售额为 50.375 亿美元，2024 年为 49.931 亿美元，同比增长 0.9%。其中中国区 2025 年净销售额为 2.791 亿美元，2024 年为 2.976 亿美元，同比下降 6.2%。中国区约占全球净销售额的 5.5%，公司风险因素表述中约称为 6%。

### 区域销售对比

| 区域 | 2025 年净销售额 | 2024 年净销售额 | 同比 |
|---|---:|---:|---:|
| 北美 | 10.330 亿美元 | 10.544 亿美元 | -2.0% |
| 拉丁美洲 | 8.812 亿美元 | 8.325 亿美元 | +5.8% |

年报称，中国区下降主要由销售量下降 10.6% 驱动，部分被客户忠诚计划带来的销售组合正面影响抵消。2025 年和 2024 年中国区没有提价。

### 对中国市场的理解

中国区目前处在“业务仍有基础，但增长承压”的状态：

- 有获批直销资格、服务网点和营养俱乐部网络，说明渠道基础仍在。
- 2025 年销售额下降，说明传统销售组织和需求转化存在压力。
- 公司持续强调数字工具、客户忠诚计划和营养俱乐部，说明未来增长重点可能从粗放招募转向客户运营、复购、服务体验和合规化触点。
- 中国健康消费、体重管理和营养补充需求长期存在，但直销行业声誉、监管敏感度和消费者信任是主要约束。

## 七、中国市场未来展望

### 1. 乐观情景

如果康宝莱中国能把营养俱乐部、官方商城、客户忠诚计划和健康管理工具打通，并严格控制营销人员宣传和计酬合规，那么它仍可能在体重管理、运动营养和社区健康服务中找到增长空间。老龄化、体重管理、轻运动和个性化营养需求会为产品提供长期需求基础。

乐观情景的前提是：产品创新有效、数字化工具提升复购、营养俱乐部能从“销售点”升级为“合规服务点”，同时不出现大规模违规宣传或变相团队计酬事件。

### 2. 中性情景

更现实的判断是，中国市场短期保持低速修复或小幅波动。康宝莱凭借品牌、服务网点、营销人员和老客户群维持基本盘，但很难回到高增长时代。数字化和客户忠诚计划会改善单客价值和留存，但未必能完全抵消直销行业整体信任成本。

中性情景下，项目更适合做“合规零售 + 服务复购 + 私域客户维护”，而不是做激进招商、裂变扩张或大规模代理招募。

### 3. 谨慎情景

如果监管继续加强对直销、社交电商、健康产品宣传、个人信息和私域营销的执法，或者出现营销人员夸大收益/功效、诱导囤货、跨区域直销、团队计酬等事件，中国区可能继续承压。公司 10-K 也提示，中国监管解释和执法存在不确定性，业务模式或服务提供商若被认定不合规，可能对中国业务造成重大不利影响。

谨慎情景下，康宝莱中国项目的最大风险不是产品本身卖不出去，而是销售组织和宣传体系失控，导致商业模式被监管或舆论重新审视。

## 八、项目合规风险清单

### A. 可做事项

| 事项 | 合规要点 |
|---|---|
| 在获批区域内销售获批产品 | 核查商务部系统区域、服务网点和产品目录 |
| 以产品零售和客户服务为核心 | 收入逻辑应来自本人向最终消费者销售产品 |
| 使用官方商城和官方工具 | 保留订单、价格、退换货和客户授权记录 |
| 做营养、运动、体重管理服务 | 避免医疗化表述，使用生活方式和营养支持话术 |
| 做客户复购和社群维护 | 明确广告属性、保护个人信息、避免收益承诺 |

### B. 不可做事项

| 高风险行为 | 风险原因 |
|---|---|
| 拉人头、按发展人数返利 | 触碰传销/变相传销风险 |
| 团队计酬、多层级抽佣 | 康宝莱中国计酬制度明确禁止团队计酬 |
| 要求购买大礼包/囤货才能加入 | 可能构成以购买商品作为加入或获利条件 |
| 夸大产品能治病、防癌、替代药物 | 违反广告、保健食品和消费者保护相关规则 |
| 跨未获批区域开展直销 | 违反直销区域监管要求 |
| 推销未纳入直销目录产品 | 超出核准产品范围 |
| 宣传“轻松赚钱”“多数人成功”“躺赚” | 涉及虚假收益宣传和直销培训监管风险 |
| 私自低价转卖、平台倒货 | 可能导致消费者权益、产品质量和渠道管理风险 |

### C. 必须核查事项

| 核查项 | 为什么重要 |
|---|---|
| 当地是否为获批直销区域 | 直销只能在批准区域开展 |
| 当地是否有服务网点 | 服务网点是消费者了解价格、退换货等服务的重要基础 |
| 具体产品是否在直销产品目录 | 官网展示产品不等于可直销产品 |
| 参与者是否具备直销员资格 | 需培训、考试、合同、直销员证等 |
| 收益是否仅来自本人直接销售 | 判断是否存在团队计酬或传销风险 |
| 营养俱乐部是否具备经营资质 | 涉及食品制售、门店经营、活动宣传等问题 |
| 社群话术是否合规 | 私域不是监管真空，聊天记录和海报都可能成为证据 |
| 是否收集健康/体重/订单等敏感数据 | 涉及个人信息保护和健康信息处理 |
| 是否有发票、售货凭证和退换货流程 | 消费者权益保护和直销监管重点 |
| 是否涉及商业贿赂或不当招待 | 直销行业和跨国公司中国业务的长期监管敏感点 |

## 九、项目判断建议

### 1. 如果你是消费者

可以把康宝莱看成营养健康消费品品牌，但不应把产品当药品使用，也不应相信“治疗疾病”“快速减重无风险”“替代医疗”的宣传。购买时应使用正规渠道，保留凭证，关注退换货条款。

### 2. 如果你考虑成为销售/业务参与者

重点看实际零售客户是否存在，而不是看上级展示的收入故事。需要问清楚：是否必须囤货、是否必须拉人、收入是否来自本人销售、是否有退货机制、是否提供发票和售货凭证、是否允许在你所在地区开展直销。

### 3. 如果你考虑合作或代理项目

不要只看“康宝莱有牌照”。应把项目拆成四张表：

- 区域表：每个城市/区县是否获批、是否有服务网点。
- 产品表：每个 SKU 是否在直销产品目录，是否有合法标签和宣称范围。
- 人员表：参与者身份、培训、合同、直销员证、服务商资质。
- 话术表：收益宣传、产品功效、招商海报、社群脚本、直播短视频文案。

只有这四张表都能闭环，项目才具备进一步评估的基础。

### 4. 如果你从投资或战略角度看中国市场

康宝莱中国不是一个可以简单按“直销扩张”估值的市场。更合理的观察框架是：

- 存量服务网络能否转成合规社区健康服务资产。
- 客户忠诚计划能否改善复购和客单价。
- 数字化工具能否降低营销人员管理和合规成本。
- 产品创新能否适应中国本土健康消费趋势。
- 公司能否避免营销人员端出现重大违规和舆情事件。
