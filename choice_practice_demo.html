导入答案 CSV
当前使用内置示例题。可以导入 JSON、Word .docx，也可以另行导入答案 CSV。
JSON 顶层必须是数组；每题需要包含 id、question、options.A/B/C/D。answer 可以是 A/B/C/D，也可以先留空。
答案 CSV 支持表头 id,answer；如果题号重复，建议使用 source,answer 或 source,id,answer。 
{
  "id": 72,
  "source": "100-2 第 72 题",
  "question": "题干内容",
  "options": {
    "A": "选项A",
    "B": "选项B",
    "C": "选项C",
    "D": "选项D"
  },
  "answer": "B"
}
